A straightforward and functional desktop calculator application built using Visual Basic .NET. This project is an educational demonstration of fundamental programming concepts and user interface design for desktop applications.

## Introduction

The Basic Calculator desktop application allows users to perform basic mathematical operations such as addition, subtraction, multiplication, and division. It's designed with simplicity in mind and provides a clear user interface, making calculations straightforward.

---

## Features
- Perform addition, subtraction, multiplication, and division operations.
- Clear inputs and reset functionality.
- Exit functionality for closing the application gracefully.
- User-friendly interface for intuitive operation.

- Visual Basic .NET

## Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/Aryosetowmn/desktopdev_kelas11semester1_port5.git
   ```
2. Open the project in **Visual Studio**.
3. Build the project to generate the executable files.
4. Run the application directly from Visual Studio or the compiled `.exe` file.

---

## Usage

1. Input two numbers in the designated text boxes.
2. Click on the respective operation button (Add, Subtract, Multiply, Divide) to perform a calculation.
3. The result will be displayed in the result field.
4. Use the **Clear** button to reset the inputs and result, or the **Exit** button to close the application.
